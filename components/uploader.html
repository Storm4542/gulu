# Uploader

使用方法

文件上传组件

拥有多文件上传并预览的功能。

   <div>
            <div>一个文件上传组件</div>
                <g-uploader
                              action="http://127.0.0.1:3000/upload"
                              name="file"
                              method="POST"
                              :parseResponse="parseResponse"
                              @addFile=addFile
                              :file-list.sync="fileList"
          
                 >
             
            </g-uploader>
	</div>
<script>
	  export default {
        name: 'uploader-demos',
        components: {GUploader, GButton},
        data() {
            return {
                fileList: [],
            };
        },
        methods: {
            addFile(file){
              this.fileList.push(file)
            },
            parseResponse(response) {
                let object = JSON.parse(response);
                return `https://image-server-gulu.herokuapp.com/upload/${object.key}`;
            },
        }
    };
</script>
属性 说明 类型 默认值
action 上传地址 String -
name 上传文件的名称 String -
method 上传方法 String POST
parseResponse 得到返回值以后的回调 String -
fileList.sync 已上传的文件列表 Array [ ]
addFile 增加文件函数 Function -
